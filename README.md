# springboot-file-upload

1. Implemented Rest API's to perform CRUD operation suc as
    upload plain text file with validations, 
    get the record with id, 
    delete the record with id.

2. API's to prferom operations:
  a. /files/upload - to upload plain text file
  b. /files/findRecordById/{id} - to fetch the record for id provided.
  c. /files//deleteRecordById/{id} - to delete the record for id provided.
